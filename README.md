# Advent of Code
Solutions for [Advent of Code](https://adventofcode.com).

To run a particular solution:
```
$ cargo run --release --bin aoc-<year>-<day>
```

To work on a solution:
```
$ cd <year>/<day>
# hack hack hack
$ cargo t
$ cargo r
```
